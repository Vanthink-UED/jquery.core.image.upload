Core Image Upload

一款轻量级上传组件

快速开始

English Doc

上传

点击按钮上传


$(".js-btn-upload").coreImageUpload ({
    url: "./upload.php",
    inputOfFile: 'image',
    uploadedCallback: function (result) {
        alert(result);
    }
});

裁剪图片并且上传

裁剪图片

<script src="../src/jquery-ui.min.js"></script>
//...
$(".js-btn-crop").coreImageUpload ({
    url: "./crop.php",
    inputOfFile: 'image',
    enableCrop: true,
    cropRatio: '1:1',
    uploadedCallback: function (result) {
        alert(result);
    }
});

如果你需要使用裁剪功能你需要引进样式jquery.core.image.min.css以及 jquery-ui.然后会带上裁剪参数
